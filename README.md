# Stop Panic Attack

A free, instant panic relief web app. No sign-up, no install — open and breathe.

**[→ Open the app](https://subrosa5.github.io/stop-panic-attack/)**

---

## What it does

Helps you get through a panic attack in real time using evidence-based techniques:

- **SOS mode** — one tap launches an immediate calming sequence with grounding reassurance
- **Breathing guide** — animated circle with 4-4-6-2 pacing (inhale, hold, exhale, hold)
- **Calming sounds** — pink noise, brown noise, binaural beats, ocean, wind, stream
- **Grounding 5-4-3-2-1** — sensory technique to bring you back to the present
- **Visual modes** — aurora, ocean, sunset and default animated canvas backgrounds
- **EN / RU** — full English and Russian interface

## How to use

Just open the link. No account, no onboarding. Tap **SOS** if you're in crisis right now, or **I need calm** to go straight to the breathing screen.

## Tech

Single HTML file. Pure vanilla JS + Web Audio API (all sounds synthesized, no audio files) + Canvas 2D for animations. Zero dependencies, zero build tools.
